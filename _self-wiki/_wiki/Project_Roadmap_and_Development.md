> This document outlines the multi-stage roadmap for a software project, detailing the progression from a basic UI to a fully AI-driven system. Each stage has defined scope and estimated timelines, providing a clear path toward the final vision.

## Evolution
- Initial stages focused on establishing a foundational user interface and core functionality.
- Subsequent stages introduce complexity, moving from rule-based systems to advanced AI integration.

## Content
### Roadmap
- **Stage 1**: Simple UI (Estimated: 2-3 weeks)
- **Stage 2**: Rule-based server + Simple UI (Estimated: 1-2 weeks) + FQA dataset integration.
- **Stage 3**: AI-based server + Improved UI (Estimated: 2-3 weeks)
- **NEXT**: Subsequent development phases.
