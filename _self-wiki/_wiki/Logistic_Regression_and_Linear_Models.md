## Content
### Linear Models vs. Logistic Regression
Linear regression models are fundamentally different from logistic regression, although both rely on linear combinations of input features. Logistic regression uses a linear combination as input to a non-linear function (the sigmoid function) to map the output into a probability between 0 and 1, making it suitable for classification tasks.

### Softmax Function
The Softmax function is a generalization of the sigmoid function, typically used in multi-class classification problems. It applies the exponential function to each input and normalizes them into a probability distribution, summing to 1.
